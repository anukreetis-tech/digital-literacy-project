## 📌 Project Overview
This project focuses on developing essential digital literacy skills required for academic and professional growth. It includes practical tasks such as creating an infographic, building a digital portfolio, exploring online platforms, practicing email etiquette, and understanding cybercrime awareness.

The aim of this project is to promote responsible, safe, and effective use of digital tools and technologies.

---

## 📂 Task Summaries

### 🖼️ Task 1: Infographic
Created a digital infographic using Canva to explain digital literacy, tools, online safety, and professional practices.

### 🌐 Task 2: Digital Portfolio
Developed and updated profiles on platforms like GitHub, LinkedIn, Kaggle, and Stack Overflow to build an online presence.

### 💻 Task 3: Platforms
Explored coding and collaboration platforms by solving a basic problem and creating a Google Form quiz.

### 📧 Task 4: Email Etiquette
Drafted professional emails and created a checklist for responsible social media usage.

### 🔐 Task 5: Cybercrime Awareness
Prepared a case study on UPI fraud and designed a checklist for staying safe online.
